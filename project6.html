Hazardous Asteroid Prediction Using Majority Voting Technique

This project developed a machine learning system to classify near-Earth asteroids as hazardous or non-hazardous, addressing a critical need for planetary defense. By combining three high-performing models via Majority Voting Technique (MVT), the solution achieved perfect accuracy, outperforming individual models and existing state-of-the-art approaches.

Data Acquisition and Preprocessing:
• Collected a Kaggle dataset of 4,687 asteroids with 40 features (e.g., velocity, diameter, orbital parameters) and binary labels (hazardous/non-hazardous).
• Removed 20 redundant features (e.g., duplicate metrics in different units) using correlation heatmap analysis, retaining 19 critical attributes like relative velocity, miss distance, and Tisserand parameter.
• Split data into 70% training (3,280 samples) and 30% testing (1,407 samples) sets after standardization.

Model Development:
• Trained six baseline models: Naive Bayes (92.24%), Logistic Regression (95.45%), Random Forest (98.86%), Decision Tree (98.86%), SVM (95.17%), and K-Nearest Neighbor (91.04%).
• Selected top 3 models (Logistic Regression, Random Forest, Decision Tree) based on higher test accuracy for ensemble learning.
• Implemented MVT to combine predictions: final classification determined by majority vote across the three models.

Results:
• MVT achieved 100% accuracy on test data, correctly classifying all 1,407 samples.
• Higher accuracy than individual models and 40% improvement over magnitude classification systems.
• Demonstrated robustness in edge cases: resolved conflicting predictions from individual models.

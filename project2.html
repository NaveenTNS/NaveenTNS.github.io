Healthcare Database System

I developed a normalized database system that centralizes all critical hospital data. This design ensures seamless access and real-time updates across all departments, greatly improving operational efficiency and patient care.

Key Steps:
• The system was implemented with 14 core entities, including Patient, Doctor, Appointment, Medical_Record, Laboratory_Tests, Billing, Inventory, Supplier, and Pharmacy
• Populated the database using 5000 records of data generated using faker and random libraries in python.
• Queried the database using SQL to test it's working.

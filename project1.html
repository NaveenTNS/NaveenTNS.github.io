Movie Review Sentiment Analysis

Built an end-to-end movie review sentiment analysis system with a Streamlit interface, Selenium scraping, and a RAG pipeline using LLaMA 3.2 and FAISS for accurate sentiment summarization.

Key Steps:
• Developed a Streamlit application to allow users to input a movie name and interact with the sentiment analysis system.
• Scraped 125 user reviews per movie from IMDB using a resilient Selenium automation pipeline.
• Preprocessed review titles using tokenization, lemmatization, and stop word removal, then generated word clouds for thematic insights.
• Generated vector embeddings of reviews using Ollama’s LLaMA 3.2 model and stored them in a FAISS Vector Database.
• Built a retriever from the vector database to efficiently query relevant review embeddings.
• Utilized Groq’s LLaMA 3.2 model to produce a sentiment summary of user reviews for the movie.
